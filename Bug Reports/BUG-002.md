# 🐞 BUG-002 - Prediction accepts invalid numeric values without validation

## Bug Information

| Item | Value |
|------|-------|
| Project | Real Estate Prediction System |
| Module | Property Prediction |
| Related Test Case | TC_PRE_003 |
| Severity | High |
| Priority | High |
| Status | Open |

---

## Environment

- Browser: Google Chrome 138
- Operating System: macOS Sequoia

---

## Preconditions

- User is on the Property Prediction page.

---

## Steps to Reproduce

1. Open the Prediction page.
2. Enter **-50** in the Area field.
3. Fill in the remaining required fields with valid values.
4. Click **Predict**.

---

## Expected Result

The system should validate numeric input and prevent prediction when the area is less than or equal to zero.

---

## Actual Result

The system accepts the invalid value and generates a prediction.
