# 🐞 BUG-006 - System allows submitting empty feedback without validation

## Overview

| Field | Value |
|-------|-------|
| Project | Real Estate Prediction System |
| Module | Contact & Feedback |
| Issue Type | Bug |
| Severity | Medium |
| Priority | Medium |
| Status | Open |
| Related Test Case | TC_CONTACT_005 |
| Related Jira Issue | REPT-87 |

---

## Environment

- Browser: Google Chrome 138
- Operating System: macOS Sequoia
- Testing Type: Manual Testing

---

## Preconditions

- User is logged in and is on the Contact page.

---

## Steps to Reproduce

1. Navigate to the Contact page.
2. Leave the Feedback field empty.
3. Click **Submit Feedback**.

---

## Expected Result

The system should display a validation message indicating that the feedback field cannot be empty and prevent form submission.

---

## Actual Result

The system allows the feedback form to be submitted without displaying any validation message.

---

## Root Cause (Assumption)

Required field validation is missing or not executed before submitting the feedback form.

---

## Notes

This defect was identified during manual functional testing as part of the Real Estate Prediction System testing portfolio.
