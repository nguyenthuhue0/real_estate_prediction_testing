# 🐞 BUG-006 - System allows submitting empty feedback without validation

## Bug Information

| Item | Value |
|------|-------|
| Project | Real Estate Prediction System |
| Module | Contact |
| Related Test Case | TC_CONTACT_005 |
| Severity | Medium |
| Priority | Medium |
| Status | Open |

---

## Environment

- Browser: Google Chrome 138
- Operating System: macOS Sequoia

---

## Preconditions

- User is logged in.
- User is on the Contact page.

---

## Steps to Reproduce

1. Navigate to the Contact page.
2. Leave the Feedback field empty.
3. Click **Submit Feedback**.

---

## Expected Result

The system should display a validation message indicating that the feedback field cannot be empty and prevent the submission.

---

## Actual Result

The system allows the feedback form to be submitted without displaying any validation message.
