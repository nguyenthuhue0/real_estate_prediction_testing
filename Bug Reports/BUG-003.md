# 🐞 BUG-003 - Price range filter returns incorrect search results

## Bug Information

| Item | Value |
|------|-------|
| Project | Real Estate Prediction System |
| Module | Property Search |
| Related Test Case | TC_SEARCH_004 |
| Severity | High |
| Priority | High |
| Status | Open |

---

## Environment

- Browser: Google Chrome 138
- Operating System: macOS Sequoia

---

## Preconditions

- User is on the Property Search page.

---

## Steps to Reproduce

1. Open the Search page.
2. Select the price range **2B – 4B VND**.
3. Click **Apply Filter**.

---

## Expected Result

Only properties priced between **2B and 4B VND** should be displayed.

---

## Actual Result

Properties outside the selected price range are displayed.
