| Module | Property Search |
| Issue Type | Bug |
| Severity | High |
| Priority | High |

| Related Jira Issue | REPT-84 |

---

## Environment

- Browser: Google Chrome 138
- Operating System: macOS Sequoia
- Testing Type: Manual Testing

---

## Preconditions

- User is on the Property Search page.

---

## Steps to Reproduce

1. Open the Search page.
2. Select a price range between **2B and 4B VND**.
3. Click **Apply Filter**.

---

## Expected Result

Only properties within the selected price range should be displayed.

---

## Actual Result

Properties outside the selected price range are included in the search results.

---

## Root Cause (Assumption)

The filtering logic does not correctly apply the selected minimum and maximum price values.

---

## Notes

This defect was identified during manual functional testing as part of the Real Estate Prediction System testing portfolio.
