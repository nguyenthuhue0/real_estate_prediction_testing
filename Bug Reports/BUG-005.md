# 🐞 BUG-005 - System accepts unsupported avatar file format upload

## Bug Information

| Item | Value |
|------|-------|
| Project | Real Estate Prediction System |
| Module | User Profile |
| Related Test Case | TC_PROFILE_006 |
| Severity | Medium |
| Priority | Medium |
| Status | Open |

---

## Environment

- Browser: Google Chrome 138
- Operating System: macOS Sequoia

---

## Preconditions

- User is logged in.
- User is on the Profile page.

---

## Steps to Reproduce

1. Navigate to the Profile page.
2. Click **Upload Avatar**.
3. Select **avatar.pdf**.
4. Click **Save**.

---

## Expected Result

The system should reject unsupported file formats and display a validation message.

---

## Actual Result

The system accepts the PDF file and uploads it successfully.
