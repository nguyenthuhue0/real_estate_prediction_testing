# 🐞 BUG-004 - Location filter returns properties from incorrect locations

## Bug Information

| Item | Value |
|------|-------|
| Project | Real Estate Prediction System |
| Module | Property Search |
| Related Test Case | TC_SEARCH_005 |
| Severity | High |
| Priority | High |
| Status | Open |

---

## Environment

- Browser: Google Chrome 138
- Operating System: macOS Sequoia

---

## Preconditions

- User is on the Property Search page.

---

## Steps to Reproduce

1. Open the Search page.
2. Select **Da Nang** as the location.
3. Click **Apply Filter**.

---

## Expected Result

Only properties located in **Da Nang** should be displayed.

---

## Actual Result

The search results include properties from other locations.
